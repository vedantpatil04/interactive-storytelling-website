# Interactive Storytelling Website

A visually rich storytelling website built using HTML, CSS, and JavaScript.

## Features

* Responsive design
* Interactive navigation
* Custom CSS animations
* Music section with controls
* Timeline section
* Editorial-style layouts
* Fully static website (no backend)

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* index.html
* style.css
* script.js
* images/

## Purpose

This project explores creative web design, storytelling layouts, animations, and user interaction using only front-end technologies.
